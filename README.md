# CIEL-_SOT_Agent

Integration attractor for the CIEL ecosystem.

This repository does **not** replace the canonical theory repository, the Omega demo cockpit, or Metatime.
It binds them through a small, explicit integration layer based on:

- repository identity,
- semantic phase,
- weighted coupling,
- Euler-style closure defect,
- machine-readable cross-reference indexing,
- multi-agent coordination on GitHub.

## Role in the ecosystem

- **canon / Seed of the Worlds** — source of truth for axioms, definitions, derivations, manifests, and nonlocal repository hyperspace
- **ciel-omega-demo** — cockpit, UI surface, educational analogies, orbital preview
- **Metatime** — historical theory and simulation source
- **CIEL-_SOT_Agent** — integration kernel, synchronizer, registry, compatibility harness, and public operational attractor

## Current structure

- `AGENT.md` — integration operating rules
- `agentcrossinfo.md` — cross-agent coordination protocol
- `docs/` — architecture, analogies, hypotheses, derivation notes, index
- `integration/` — repository registry, coupling map, hyperspace index, reports
- `src/ciel_sot_agent/` — executable synchronization code
- `scripts/` — convenience launchers
- `tests/` — numerical sanity tests

## First executable kernel

The first mandatory executable component is the **repository phase synchronizer**.
It reads a registry of repository identities and couplings and emits:

- weighted Euler vector,
- closure defect,
- pairwise phase tensions,
- a machine-readable synchronization report.
