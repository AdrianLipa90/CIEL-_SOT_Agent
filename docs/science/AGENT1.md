# AGENT1.md

## Folder role

`docs/science/` stores compact scientific notes and derivation bridges used by Agent 1.

These files are allowed to support integration logic,
but they are not automatically equivalent to fully canonical proof objects.

---

## Agent 1 scope in this folder

Agent 1 may use this folder for:

- hypotheses,
- derivation bridges,
- imported-anchor notes,
- explicit upstream/downstream references into registries and code.

---

## Rules

- Every file here should remain linked to the machine registry.
- Imported vs native provenance should remain explicit.
- Derivation notes must not silently become engine claims.
- If a scientific claim lacks executable validation, that limit should remain visible.
