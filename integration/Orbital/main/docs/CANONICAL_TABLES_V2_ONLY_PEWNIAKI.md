# CANONICAL TABLES V2 — ONLY PEWNIAKI

Ta wersja zawiera wyłącznie rekordy oznaczone jako:
- `confirmed`
- `confirmed-mechanism`

Bez `candidate`, `unresolved` i bez wartości roboczych niepodpartych plikami.

## Zawartość
- formal symbols
- couplings only-pewniaki
- runtime symbol ↔ file/function map only-pewniaki
- dependency chain only-pewniaki
- emergent values policzone bez domysłów z kontraktu startowego

## Uwaga
`A_ij` pozostaje notacją toy-modelową dla efektywnego pair-kernelu.
Jest utrzymane jako `confirmed-mechanism`, bo odpowiada już potwierdzonemu mechanizmowi
budowanemu z `A`, `W_ij`, `F_ij`, `tau_i`, `gamma_i`.
