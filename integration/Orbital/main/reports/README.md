# Orbital Reports

## Role

This folder stores generated report artifacts emitted by the orbital integration runtime.

## Expected report path

- `global_orbital_coherence_pass/summary.json`
- `global_orbital_coherence_pass/summary.md`
- `global_orbital_coherence_pass/real_geometry.json`

## Rule

These reports are derived diagnostics from `integration/Orbital/main/global_pass.py` and should be treated as generated state, not as primary architectural truth.
