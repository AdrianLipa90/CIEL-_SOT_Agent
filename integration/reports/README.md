# Integration Reports

## Role

This folder stores generated report artifacts emitted by the SOT integration layer.

## Current report objects

- `initial_sync_report.json` — baseline synchronization report for the discrete repository phase model.
- `live_gh_coupling_report.json` — live GitHub coupling report refreshed by the operational automation path when enabled and executed.

## Structural rule

Reports in this folder are derived artifacts.
They should remain traceable to explicit code, registry, and workflow paths rather than acting as undocumented sources of truth.
